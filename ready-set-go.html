Ready, set, Go!|

Hi! This is my first blog post. I've been learning about the Go programming language lately and thought it was an interesting topic to start with.

Golang (as known as Go) is a free and open source being developed by Google development since 2007. In 2012, the language reached 1.0 milestone.

Features

Go is being designed and implemented by brilliant minds like Ken Thompson and Rob Pike, among others, in an attempt to fill the gaps where Python, Java and C++ have room for improvement.

It aims to mix some of the best features from these and other modern programming languages: clear and concise syntax but still statically typed and compiled directly to machine code, garbage collection, concurrency, etc. For example, you don't have to necessarily declare the type of a variable, Go uses type inference for that.

It also has some kind of object-orientation with functional programming features, but no object hierarchy or classes.
